Benvenuto nell'applicazione Spring Boot!

Questa è un'applicazione web dimostrativa realizzata con Spring Boot, Spring Security e Bootstrap. Esplora le funzionalità di autenticazione e gestione delle aree pubbliche e private.

Funzionalità principali:

  • Sistema di autenticazione con Spring Security
  • Aree pubbliche e private
  • Design responsive con Bootstrap 5
  • Template Thymeleaf per rendering server-side
